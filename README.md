#NinjaGram

##Proyecto de práctica realizado en cordova (necesitas tener instaladas las variables de entorno de JAVA y GRADLE).

**El proyecto no está compilado, por lo que para probarlo en el navegador deberás ejecutar CORDOVA RUN BROWSER*
y para jecutarlo en Android Studio deberás ejecutar CORDOVA RUN ANDROID*

*Utilizar minúsculas

Además, el proyecto está conectado con firebase.
